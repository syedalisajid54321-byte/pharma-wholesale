## 🚀 Quick Setup Guide (30 Minutes to Running App)

### Step 1: Install Flutter (if not installed)
```bash
# Download Flutter from https://flutter.dev/docs/get-started/install
# Add to PATH and verify:
flutter doctor
```

### Step 2: Install Firebase CLI
```bash
npm install -g firebase-tools
firebase login

# Install FlutterFire CLI
dart pub global activate flutterfire_cli
```

### Step 3: Create Firebase Project
1. Go to https://console.firebase.google.com
2. Click "Add project" → Name it **pharma-wholesale**
3. Enable Google Analytics (optional)

### Step 4: Enable Firebase Services
In Firebase Console:

**Authentication:**
- Go to Build → Authentication → Get Started
- Enable **Email/Password** sign-in method
- Create your admin user: Authentication → Users → Add User
  - Email: `admin@pharma.com`
  - Password: (choose a strong password)

**Firestore Database:**
- Go to Build → Firestore Database → Create database
- Choose **Production mode**
- Select your nearest region

### Step 5: Connect Flutter to Firebase
```bash
cd pharma_wholesale
flutterfire configure --project=pharma-wholesale
```
This auto-generates `lib/firebase_options.dart` with your credentials.

### Step 6: Deploy Firestore Rules & Indexes
```bash
firebase init firestore
# Use existing files when prompted

firebase deploy --only firestore
```

### Step 7: Run the App
```bash
flutter pub get
flutter run
```

### Step 8: Build APK
```bash
# Debug APK (for testing):
flutter build apk --debug

# Release APK (for distribution):
flutter build apk --release

# APK location: build/app/outputs/flutter-apk/app-release.apk
```

| Feature | Description |
|---------|-------------|
| 🔐 Login | Firebase Auth with email/password |

## 🔧 Troubleshooting

**Build fails with Firebase error:**
- Make sure `google-services.json` is in `android/app/`
- Run `flutterfire configure` again

**Firestore permission denied:**
- Deploy rules: `firebase deploy --only firestore:rules`
- Make sure user is logged in

**Font not loading:**
- Run `flutter pub get`
- Check internet connection (Google Fonts downloads on first run)
